Was ist Hypertext?

Hypertext ist Text, der Verweise (Links) auf andere Texte enthält. HTML steht für Hypertext Markup Language — eine Auszeichnungssprache, die Struktur und Verweise beschreibt.

Im Gegensatz zu einem Buch kann man in Hypertext nicht-linear navigieren: Jeder Link führt zu einem anderen Dokument oder einer anderen Stelle.
